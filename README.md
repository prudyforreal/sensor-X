# sensor-X

 Built a simple security system using an Arduino Uno and a PIR sensor. The system detects motion and triggers a red LED and buzzer for alerts, while a green LED indicates No movement. Programmed and simulated using Tinkercad to demonstrate real-time sensor input and output control.
